This folder contains certification logos for your LinkedIn certifications.

To add your certification logos:
1. Download the logo from the certification provider or use a screenshot
2. Save it as a PNG or JPG file with a descriptive name
3. Place it in this folder
4. Update the path in the certifications array in the script.js file

Sample logos to acquire:
- Google (for Google certifications)
- EF Set (for English certifications)
- Android (for Android Enterprise certifications)
- Coursera
- LinkedIn Learning
- Microsoft
- AWS
- etc.

Default dimension: 60x60 pixels
